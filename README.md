Java and cloud certification study notes.

## Sections

- [Oracle Java Silver](./oracle-java-silver/) - Java Silver notes and reference PDF.

## Organization

- Each certification/project keeps its notes and resources inside its own directory.
- `notes/` contains Markdown study notes grouped by content area and ordered by learning sequence.
- `resources/` contains PDFs and other reference files.
- File names use lowercase kebab-case. Ordered notes start with a two-digit prefix.
